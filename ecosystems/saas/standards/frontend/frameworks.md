# Frameworks e Bibliotecas Frontend

## Bootstrap 5

O projeto utiliza Bootstrap 5 como framework CSS principal.

### Diretrizes de Uso

- Priorizar o uso de classes do Bootstrap ao estilizar elementos HTML
- Criar classes personalizadas apenas quando não for possível atingir o resultado desejado utilizando as classes utilitárias do Bootstrap
- Evitar sobrescrever estilos do framework com excesso de overrides
- Manter consistência visual utilizando o sistema de design do Bootstrap

## jQuery

O projeto faz uso do jQuery para manipulação do DOM e interações.

### Diretrizes de Uso

- Sempre que possível, preferir utilizar jQuery ao invés de JavaScript puro para manipulações no DOM e interações
- Manter consistência com o padrão já estabelecido no projeto
- Utilizar métodos jQuery para seleção, manipulação e eventos
